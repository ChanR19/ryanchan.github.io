PROJECTS ABOUT ME

Prosthetic Control Interface

During the school year, I worked on a project team in partnership with a prosthetics startup to develop a low-cost, non-invasive, and wireless sEMG armband that would be used to control prosthetic limbs.

On this project, I designed most of the printed circuit board and did so using Autodesk EAGLE. Additionally, I wrote a large part of the software using the C programming language and Texas Instrument's CCStudio IDE.
